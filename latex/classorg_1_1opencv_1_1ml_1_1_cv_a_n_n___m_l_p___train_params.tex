\subsection{Detailed Description}
Parameters of the M\+LP training algorithm. You can initialize the structure by a constructor or the individual parameters can be adjusted after the structure is created.

The back-\/propagation algorithm parameters\+:

Strength of the weight gradient term. The recommended value is about 0.\+1.

Strength of the momentum term (the difference between weights on the 2 previous iterations). This parameter provides some inertia to smooth the random fluctuations of the weights. It can vary from 0 (the feature is disabled) to 1 and beyond. The value 0.\+1 or so is good enough {\ttfamily }
